    GitHub Activity Mining Project

    GitHub Activity Mining Project tries to visualise and predict activity of GitHub projects. GHAM uses scikit-learn for making prediction analysis. It downloads information about forks, pull requests, commits, comments, issues and how they changed over time and estimates how many percent of commits remain. It then convert this information into four classes:

    • if remaining percentage is above 66 %, it shows At the Beginning tag,
    • if remaining percentage is between 34 % and 66 %, it shows Active tag,
    • if remaining percentage is between 6 % and 33 %, it shows Declining tag,
    • if remaining percentage is below 6 %, it shows Probably Dead tag.

    You can obtain badge for your README.md for GitHub. Just see the repository detail for Markdown tag.

    Machine learning model for GHAM

    Used machine learning method is Gradient Tree Boosting for regression. Based on ten-fold cross-validation, root-mean-square error, while predicting value from 0 to 100, is ~13.
